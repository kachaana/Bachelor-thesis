void CToggl::on_ButtonStart_clicked ( )
slot

CToggl::on_ButtonStart_clicked Function will initiate all processes that shoul be triggered after button Start/Stop pushed.

  • changes of icons (red/green)
  • sending of time entries
  • updating of changes in ".conf" file
  • start/stop timer
    Parameters
    actionCan be BUTTONPUSHEd for manual stop/start. DESKTOPCHANGED behaves the same but doesn't change icons.
